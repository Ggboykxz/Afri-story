## 🇫🇷 À Propos

**AfriStory** est une plateforme numérique panafricaine dédiée à la publication et découverte de contenus numériques africaine:

- **Webtoons** - Bandes dessinées numériques à défilement vertical
- **BD** - Bandes dessinées traditionnelles
- **Romans illustrés** - Romans avec illustrations
- **Contenu hybride** - Combination des formats

La plateforme met en valeur les créateurs Africans et Afro-descendants avec un système de monétisation équitable.

### Vision

Devenir la référence continentale pour le contenu numérique africain, en offrant:
- Une vitrine mondiale aux créateurs Africans
- Un écosystème de monétisation juste (70%+ aux artistes)
- Une communauté engaged et supportive

## 🚀 Installation Locale

```bash
# Cloner
git clone https://github.com/Ggboykxz/Afri-story.git
cd Afri-story

# Installer
npm install

# Configurer
cp .env.example .env.local

# Développement
npm run dev

# Build
npm run build

# Lint
npm run lint
```
