# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-07

### Added
- Initial release of AfriStory platform
- User authentication (Google, Email)
- Role-based system (visitor, reader, artist_draft, artist_pro, artist_mentor, enterprise, moderator, supervisor, admin)
- Webtoon and BD reader
- Create work functionality for artists
- Profile editing (name, bio)
- Navigation system (mobile bottom nav, desktop sidebar)
- Firebase hosting deployment
- Vercel deployment
- Cloudinary integration for image uploads
- Firestore security rules
- Dark mode support
- Mobile hamburger menu

### Features
- Home page with trending works
- Explore page with filters
- Library for favorites and reading history
- Forum system with public and premium sections
- Rankings (Top works)
- Subscription plans (Standard, Premium, Supporter)
- AfriCoins system for micro-payments
- Book clubs
- Contests
- AMA sessions
- Collaboration hub
- Notifications system
- Moderation dashboard
- Admin dashboard
- Artist dashboard with statistics

### Technical Stack
- React 19
- TypeScript 5.8
- Vite 6
- Tailwind CSS 4
- Framer Motion
- Firebase (Auth, Firestore, Storage)
- Cloudinary
